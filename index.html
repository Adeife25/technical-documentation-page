JavaScript and Java

Introduction

JavaScript is a popular programming language used to create dynamic and interactive websites. It runs in the browser and allows developers to enhance user experience by responding to events, manipulating content, and more.

Variables and Data Types

Variables store data values. In JavaScript, you can declare variables using var, let, or const. Common data types include:

  • String "Hello"
  • Number 42
  • Boolean true or false
  • Null and Undefined
  • Object and Array

js let name = "Ada"; const age = 25;

Functions

Functions are blocks of reusable code. You can define them using the function keyword or as arrow functions.

Loops

Loops let you run a block of code multiple times.

  • for loop
  • while loop
  • for...of loop

Arrays and Objects

Arrays store lists of data, while objects store key-value pairs.

DOM Manipulation

DOM (Document Object Model) allows JavaScript to change HTML and CSS on the page.

Conclusion

JavaScript adds interactivity and power to web pages. Mastering the basics like variables, functions, and the DOM prepares you for advanced web development.

HTML for Beginners

Introduction

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It forms the foundation of all websites by defining elements like headings, paragraphs, images, links, and more.

Elements and Tags

HTML is made up of elements, which are wrapped in tags. Tags tell the browser how to display the content. Every HTML document starts with a <!DOCTYPE html> declaration followed by the <html> tag.

  • <h1> to <h6>Headings
  • <p>Paragraphs
  • <a>Links
  • <img>Images
  • <div>Containers

Attributes

Attributes provide additional information about elements. They're always included in the opening tag and follow a name="value" format. Common attributes include:

  • href for links
  • src for images
  • alt for alternative text
  • class
CSS Layout Techniques

Introduction

CSS Layout Techniques help control how elements are arranged on a webpage. These techniques determine the structure, responsiveness, and visual flow of a website. With proper layout, you can build pages that work on all screen sizes.

Box Model

Every HTML element is treated as a box in CSS. The Box Model includes four parts: content, padding, border, and margin. Understanding how these parts interact is crucial for spacing and alignment.

Display Property

The display property defines how an element appears on the page. Common values include:

  • blockTakes up full width
  • inlineTakes up only as much width as necessary
  • inline-blockLike inline, but allows setting width/height
  • noneHides the element

Positioning

CSS offers several positioning options to place elements precisely:

  • staticDefault position
  • relativePositioned relative to its normal spot
  • absolutePositioned relative to its closest positioned ancestor
  • fixedStays in place even when scrolling
  • stickyScrolls until it reaches a defined position

Flexbox

Flexbox is a powerful layout tool that makes it easy to align, space, and distribute elements within a container. It works in one direction (row or column) and is great for building responsive layouts.

  • display: flexEnables Flexbox on a container
  • justify-contentAligns items horizontally
  • align-itemsAligns items vertically
  • flex-wrapAllows items to wrap into multiple lines

Grid

CSS Grid Layout allows two-dimensional layout control — both rows and columns. It's ideal for complex designs and full page layouts.

  • display: grid Enables Grid on a container
  • grid-template-columns and grid-template-rowsDefine grid structure
  • gapAdds spacing between grid items
  • grid-column and grid-rowPlace items across the grid

Media Queries

Media queries allow you to apply different styles based on screen size or device type. This is essential for creating responsive layouts that work on phones, tablets, and desktops.

Example:

@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}

Conclusion

CSS Layout Techniques such as the box model, Flexbox, Grid, and media queries are key to building modern, responsive websites. Mastering them gives you the power to control layout, alignment, and responsiveness with precision.

Git and GitHub Workflow

Introduction

Git is a version control system that tracks changes in code projects, while GitHub is a cloud-based platform for hosting Git repositories. Together, they help developers collaborate, manage versions, and contribute to open-source or team projects.

Installing Git

To get started, you need to install Git on your machine. You can download it from git-scm.com. After installation, set up your identity:

git config --global user.name "Your Name"
git config --global user.email "your@email.com"

Basic Git Commands

  • git initInitialize a new Git repository
  • git add .Stage all changes
  • git commit -m "Message"Commit changes with a message
  • git statusCheck the state of your working directory
  • git log View commit history

Working with GitHub

After creating a repository on GitHub, you can connect your local project to it:

git remote add origin https://github.com/yourusername/repository-name.git
git push -u origin main

GitHub Workflow

A typical GitHub workflow includes:

  1. Fork or clone the repository
  2. Create a new branch: git checkout -b feature-name
  3. Make changes and commit them
  4. Push the branch to GitHub: git push origin feature-name
  5. Open a Pull Request (PR) on GitHub
  6. Get your changes reviewed and merged into the main branch

Branches

Branches help you work on new features or fixes without affecting the main codebase. Once you're done, you can merge your branch into main using a pull request.

Conclusion

Understanding Git and GitHub workflow is essential for modern web development. It helps you collaborate with others, maintain clean code history, and confidently manage changes in any project.

Python Introduction

Introduction

Python is a high-level, interpreted programming language known for its readability and simplicity. It's widely used for web development, data analysis, automation, artificial intelligence, and more. Python's syntax allows developers to express concepts in fewer lines of code than many other languages.

Variables and Data Types

Python uses variables to store data, and the type of data can be automatically inferred. Common data types include:

  • String - text (e.g., "Hello")
  • Integer - whole numbers (e.g., 42)
  • Float - decimal numbers (e.g., 3.14)
  • Boolean - True or False
  • List - ordered sequence (e.g., [1, 2, 3])
  • Dictionary - key-value pairs (e.g., {"name": "Ada"})

Example:
name = "Ada"
age = 25
is_student = True

Functions

Functions are defined using the def keyword and allow you to reuse blocks of code.

Example:

def greet(name):
  print("Hello, " + name)

Loops

Loops in Python are used to repeat actions. The most common types are:

  • for loop - iterates over sequences
  • while loop - runs while a condition is true

Conditional Statements

Conditional statements control the flow of a program based on conditions using if, elif, and else.

Example:

if age > 18:
  print("Adult")
else:
  print("Minor")

Conclusion

Python is beginner-friendly and versatile. Learning its basics—variables, functions, loops, and conditions—forms the foundation for building powerful and efficient applications across many domains.
